# GrippitzPoison

Displays remaining poison duration on weapons for Rogues.

## Information

- **Version:** 1.6-TBC
- **CurseForge:** https://www.curseforge.com/wow/addons/grippitzpoison

## Installation

1. Download the latest release
2. Extract to `World of Warcraft\Interface\AddOns\`
3. Restart WoW
